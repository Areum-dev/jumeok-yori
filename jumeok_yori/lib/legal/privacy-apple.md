# Apple App Privacy 작성본

**버전:** 1.0.0
**작성일:** 2026-07-13
**앱명:** 주먹요리

이 문서는 App Store Connect의 App Privacy 섹션 작성을 위한 참고 자료입니다.

---

## 개인정보 처리 방침 URL

앱 내 설정 > 개인정보처리방침에서 확인 가능합니다.

---

## 데이터 수집 여부

앱이 데이터를 수집합니다: **예**

---

## 수집하는 데이터 유형 (App Store Connect 항목별)

### 연락처 정보 (Contact Info)
- **이메일 주소**: 수집함
  - 사용 목적: 앱 기능 (계정 생성 및 로그인), 사용자 ID
  - 타사 공유: 아니오
  - 추적 사용: 아니오

### 위치 (Location)
- **정확한 위치 (Precise Location)**: 수집함
  - 사용 목적: 앱 기능 (주변 음식점 추천)
  - 타사 공유: 아니오
  - 추적 사용: 아니오
  - 백그라운드 수집: 아니오 (포그라운드만)

### 식별자 (Identifiers)
- **사용자 ID**: 수집함 (익명 UUID)
  - 사용 목적: 앱 기능 (비로그인 사용자 서비스 제공)
  - 타사 공유: 아니오
  - 추적 사용: 아니오

### 사용 데이터 (Usage Data)
- **제품 상호 작용**: 수집함 (추천 이력, 즐겨찾기)
  - 사용 목적: 앱 기능 (기록 관리), 앱 기능 개선
  - 타사 공유: 아니오
  - 추적 사용: 아니오

---

## 추적 여부

이 앱은 Apple의 App Tracking Transparency 프레임워크 상 "추적"에 해당하는 데이터를 수집하지 않습니다.

---

## 개인정보 관련 링크

- **개인정보처리방침**: 앱 내 설정에서 확인
- **개인정보 삭제 요청**: 앱 내 문의하기 > 개인정보 문의

---

## App Store Connect 작성 시 참고 사항

App Store Connect > 앱 > App 개인정보 보호 섹션에서 위 내용을 바탕으로 작성합니다.

1. 개인 정보 처리 방침 URL 입력 (앱 내 제공됨을 명시)
2. 데이터 수집 여부: 예
3. 이메일 주소: 앱 기능 > 앱 기능 체크
4. 정확한 위치: 앱 기능 > 위치 공유 체크
5. 사용자 ID (익명): 앱 기능 > 분석 체크
