## 1. Google Play Console에 입력할 URL

| 용도 | URL | Play Console 입력 위치 |
|---|---|---|
| 운영 웹사이트 | `https://{DOMAIN}` | 스토어 등록정보 > 연락처 세부정보 > 웹사이트 |
| 개인정보처리방침 | `https://{DOMAIN}/privacy` | 앱 콘텐츠 > 개인정보처리방침 |
| 이용약관 | `https://{DOMAIN}/terms` | (선택) 스토어 등록정보 설명에 링크로 기재 |
| 계정 삭제 | `https://{DOMAIN}/delete-account` | 앱 콘텐츠 > 데이터 보안 > "계정 삭제 웹페이지" |
| 고객지원 | `https://{DOMAIN}/support` | 스토어 등록정보 > 연락처 세부정보 > 이메일/웹사이트 |
| 문의 이메일 | `1vpdrnls@gmail.com` | 스토어 등록정보 > 연락처 세부정보 > 이메일 |

추가로 만든 보조 페이지 (필수는 아니지만 참고용으로 등록 가능):

- 위치정보 이용약관: `https://{DOMAIN}/location-policy`
- 마케팅 정보 수신 동의: `https://{DOMAIN}/marketing-consent`
- 사장님(사업자) 이용약관: `https://{DOMAIN}/business-terms`
- 커뮤니티 운영 정책(신고 절차 포함): `https://{DOMAIN}/community-guidelines`
- 청소년 보호 정책: `https://{DOMAIN}/privacy/children`

모든 URL은 로그인이 필요하지 않으며, 모바일/PC 모두에서 읽을 수 있는 일반 HTML 페이지입니다.

## 2. 데이터 보안(Data Safety) 섹션 작성 참고

원본 출처: `jumeok_yori/lib/legal/data-safety-google.md` (기존에 작성해 두신 문서를 그대로 반영)

### 데이터 수집 여부: 예

| 데이터 유형 | 수집 | 목적 | 필수 여부 | 처리 방식 |
|---|---|---|---|---|
| 정밀 위치 | 수집함 | 주변 음식점 추천 | 필수 아님 (거부 시 기본 위치로 대체) | 서버 전송 |
| 이메일 주소 | 수집함 | 계정 관리, 로그인 | 필수 | 서버 전송 |
| 앱 내 검색/추천 기록 | 수집함 | 추천 기록 관리, 통계 | 필수 | 서버 전송 |
| 사진(메뉴/가게/사업자등록증 이미지) | 수집함(사장님 기능 이용 시) | 가게/메뉴 정보 제공 | 선택 | 서버(Supabase Storage) 전송 |

### 데이터 공유

제3자와 공유하지 않습니다.

### 보안 관행

- 전송 중 암호화: 예 (HTTPS/TLS)
- 저장 데이터 암호화: 예 (Supabase, 비밀번호 단방향 해시)
- 데이터 삭제 요청 가능: 예 (앱/웹 내 계정 삭제 또는 `/delete-account` 페이지, 이메일 문의)

### 위치정보 사용 목적

주변 음식점까지의 거리를 계산해 조건에 맞는 메뉴를 추천하기 위함입니다. 포그라운드(앱/웹 사용 중)에서만
사용하며 백그라운드 상시 수집은 하지 않습니다. 브라우저/앱 권한을 거부해도 기본 위치 또는 직접 지정한
기준 위치로 서비스를 계속 이용할 수 있습니다.

### 사진/이미지 사용 목적

- 사장님이 등록하는 가게 대표 이미지, 메뉴 사진 (서비스 노출용, `menu-images` 공개 버킷)
- 사장님 가게 등록 심사를 위한 사업자등록증 이미지 (비공개, `business-licenses` 버킷, 본인과 관리자만 조회 가능)

### 계정 생성 방식

이메일 + 비밀번호 (Supabase Auth). 소셜 로그인은 사용하지 않습니다. 이메일 인증(Confirm email)이
활성화되어 있다면 가입 후 인증 메일 확인이 필요합니다.

### 계정 삭제 방식

- 앱: 마이페이지 > 설정 > 회원탈퇴 (Flutter 앱, 기존 구현)
- 웹: `/delete-account` 페이지에서 로그인 후 본인 확인 절차를 거쳐 즉시 삭제 (이번에 새로 구현,
  service role을 사용하는 서버 API가 실제로 Supabase Auth 계정을 삭제합니다)
- 이메일: `1vpdrnls@gmail.com` 로 삭제 요청 메일 발송 (제목 자동 입력됨)

### 광고 포함 여부

**확인 필요.** 코드베이스(Flutter 앱, 웹)에서 광고 SDK나 광고 관련 코드를 발견하지 못했습니다.
현재 시점 기준으로는 광고가 포함되어 있지 않은 것으로 판단되나, 최종 확인은 운영자가 해주세요.

### 금융 기능 포함 여부

없음. 결제, 송금, 주문/배달 중개 기능은 코드에 존재하지 않습니다 (이용약관에도 "결제·배달 기능 없음"
으로 명시되어 있습니다).

### 건강 기능 포함 여부

없음.
